# README 截图指南

本指南用于手动生成 GitHub README、Release 和项目介绍所需的脱敏截图资产。项目当前没有引入自动截图依赖，保持手动截图即可，避免为发布材料增加额外运行环境。

## 准备环境

1. 安装依赖并启动 PC Web：

   ```bash
   npm install
   npm start
   ```

2. 打开 <http://localhost:8787/>。
3. 将主题切换为深色。
4. 开启展示模式，确认顶部出现“展示模式已开启”。
5. 截图前关闭系统通知，并再次确认页面中没有真实用户名、商户、金额、API Key 或其他个人信息。

## 开启展示模式

可以使用任意一种方式：

- 点击 PC 顶部工具栏中的“展示模式”。
- 无数据时点击首次使用引导中的“开启展示模式”。
- 直接访问 <http://localhost:8787/?display=1>。

展示模式只替换页面呈现，不修改真实账本。需要完整可交互数据时，可以另外访问 <http://localhost:8787/?demo=1> 加载独立演示账本；演示数据固定保存在 `ledgermind-demo` 用户下，可通过页面顶部“清除演示账本”单独删除。
