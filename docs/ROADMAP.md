# Roadmap

## 已完成的发布基础

- 待执行草稿箱与 24 小时重新确认
- 脱敏操作审计
- 数据质量评分与修复建议
- 月度、预算、订阅、数据质量和导出记录报告中心
- 智能体安全边界、Docker Compose 与 GitHub Actions CI

## 近期

- 支持更多微信、支付宝和银行卡账单变体
- 增强导入错误诊断与可恢复流程
- 扩展本地模型和 OpenAI-compatible 服务配置
- 完善 Docker 本地部署
- 月报导出为图片 / Markdown

## 中期

- 可选的端到端加密多端同步
- OCR 票据和发票识别
- 更细粒度的预算提醒与规则模板
- 更完整的规则引擎与可复用记账模板
- 演示数据和截图资产自动更新

## 长期

- 多账本与家庭共享模式
- 权限、审计和隐私控制中心
- 插件化账单解析器与 Agent 工具
- 插件系统与记账模板市场
- 更完整的数据权限、脱敏和隐私控制
- 完整的自托管发布流程

Roadmap 不构成发布时间承诺，优先级会根据实际使用反馈调整。
