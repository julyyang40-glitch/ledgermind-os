# LedgerMind OS v0.1.0-beta

> 首个公开测试版本

LedgerMind OS 是一个本地优先的中文自动记账智能体 OS。本版本面向个人用户和开源开发者开放测试，提供 PC Web 与 Android APK，覆盖账单导入、交易分类、人工审核、预算分析、月度报告和中文财务问答等主要流程。

这是 Beta 版本，适合本地体验、功能验证和二次开发。涉及重要财务数据时，请先完成备份，并保留原始账单文件。

## 核心亮点

- 导入微信、支付宝、CSV 和 XLSX 账单
- 使用规则、商户记忆和置信度完成自动分类
- 通过智能审核箱集中处理待确认交易
- 提供预算状态、账本健康度、消费图表和中文月度报告
- 支持 SiliconFlow / DeepSeek V4 Flash 财务问答
- 云端模型不可用时回退到本地账本分析
- Android 端支持不依赖 PC 后端的本地离线账本
- 提供展示模式、独立演示账本、JSON 备份和 CSV 导出
- 修改建议进入待执行草稿箱，并由用户确认
- 操作审计、数据质量修复建议和报告中心形成可追踪的 Agent 闭环

## PC Web 功能

- 财务指挥中心：展示月度收支、净额、分类率、待确认数量和节省空间
- 账单导入：识别微信、支付宝、CSV、XLSX、编码、表头和重复交易
- 交易管理：搜索、筛选、分类、批量修改、确认、忽略和删除
- 智能审核箱：聚合低置信度、大额消费、预算风险和周期扣费事项
- 预算与分析：分类预算、使用进度、每日趋势、Top 商户和分类占比
- 账本健康度：结合分类完整度、待确认状态和预算风险生成状态说明
- 月度报告：生成交易数、收入、支出、净额、主要分类和自然语言摘要
- 自动化规则与记忆：支持中文规则创建和分类修正学习
- 系统诊断：显示后端、模型、账本、预算、规则、智能体和导入状态
- 展示模式与 Showcase：用于脱敏演示和 GitHub README 截图

## Android APK 功能

- 移动端首页、交易、洞察、智能体和设置分区
- 本地保存账本，无需持续运行 PC 后端
- 支持手工记账和账单文件导入
- 支持交易分类、状态确认、预算和月报
- 支持本地账本分析与可选云端模型调用
- 支持 JSON 备份和 CSV 导出
- 适配 Android 安全区、底部导航和常见窄屏尺寸
- 支持深色与浅色主题

## AI 智能体能力

- 使用中文回答月度花销、分类、订阅和预算问题
- 回答包含结论、重点发现、建议操作、数据来源和模型来源
- 可根据自然语言创建部分自动分类规则
- 结合当前账本生成本地分析结果
- 可选接入 SiliconFlow OpenAI-compatible API 和 DeepSeek V4 Flash
- 未配置 API Key、请求失败或超时时使用本地分析回退

智能体输出仅用于个人账本整理和辅助分析，不构成投资、税务、会计或其他专业财务建议。

## 本地优先和隐私能力

- PC 端账本默认保存在本机 `data/ledger-store.json`
- Android 端离线账本默认保存在当前设备
- API Key 由用户自行配置和保管，仓库不包含真实 Key
- 支持当前用户 JSON 备份和交易 CSV 导出
- 展示模式会脱敏用户名、商户和金额呈现
- 演示账本使用独立的 `ledgermind-demo` 用户，不覆盖真实账本
- 支持单独清除演示账本和清空当前用户数据
- 草稿、审计和导出记录按真实用户、演示账本和展示模式隔离
- 审计不记录 API Key、账单原文或完整导入文件
- 智能体不会直接修改账本，导入批次暂不支持撤销

使用云端模型时，问题和必要的账本摘要可能发送给对应模型服务商。请在配置前阅读服务商条款，并避免输入不必要的敏感信息。

- 原有业务测试 `16/16` 继续通过；新增安全工具测试后总计 `24/24`
- 真实微信 XLSX 账单 `98/98` 行导入成功
- 微信、支付宝示例账单解析通过
- 分类修正学习、批量分类、预算和状态同步测试通过
- 展示模式和独立演示账本流程可用
- `412×915` 与 `390×844` 移动端视口无横向溢出
- Android Debug APK 构建通过
- Android APK Signature Scheme v2 签名校验通过

## 已知限制

- 当前为单机、本地优先版本，尚未提供正式的多端实时同步
- 不同时间或地区导出的账单格式可能存在差异，尚不能保证兼容所有变体
- Android 提供的是 Debug APK，公开分发前仍需配置正式签名和发布流程
- 云端智能体依赖用户自行配置的 API Key、网络和第三方模型服务可用性
- 本地分析主要基于规则和当前账本统计，能力范围小于完整云端模型
- 暂未提供 OCR 票据识别、银行卡账单全覆盖和 Docker 官方镜像
- 数据清空后无法自动恢复，执行前需要自行验证备份
- 智能体建议可能存在遗漏或误判，重要交易和预算决策需要人工确认
